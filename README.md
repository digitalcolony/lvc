# Lee Van Cleef: The Bad & The Best

A tribute website dedicated to Lee Van Cleef, the iconic "Prince of Villains" of Spaghetti Western cinema. This project showcases his most memorable roles, distinctive style, and contribution to the genre.

## Features

*   **Immersive Design**: A "Western" aesthetic using texture overlays, custom fonts (Rye, Playfair Display, Lora), and a curated color palette (Gunmetal, Gold, Leather, Blood Red).
*   **Responsive Layout**: Fully responsive design that adapts to mobile, tablet, and desktop screens.
*   **Animations**: Smooth fade-in effects on scroll and interactive hover states for cards.
*   **Role Gallery**: Detailed cards for his iconic characters (Angel Eyes, Colonel Mortimer, Sabata).
*   **SEO Optimized**: Includes meta tags for search engines and Open Graph/Twitter cards for social sharing.

## Technologies Used

*   **HTML5**: Semantic structure.
*   **CSS3**: Custom properties (variables), Flexbox, Grid, and CSS animations.
*   **JavaScript (Vanilla)**: Intersection Observer API for scroll animations and smooth scrolling behavior.

## How to Run

1.  **Clone or Download** the repository.
2.  **Open `index.html`** in your preferred web browser.
    *   *Tip*: For the best experience during development, use a local server (e.g., VS Code "Live Server" extension) to handle local file paths correctly, though this project works fine directly from the file system.
